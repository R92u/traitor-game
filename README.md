# 🕵️ ПРЕДАТЕЛЬ — Мультиплеер

Социальная игра на доверие. Несколько устройств, одна сеть или интернет.

## Структура

```
traitor-game/
├── server.js          ← Node.js сервер (Socket.io)
├── package.json
└── public/
    └── index.html     ← Весь фронтенд (один файл)
```

---

## 🚀 Запуск локально

```bash
# 1. Установить зависимости
npm install

# 2. Запустить сервер
npm start
# или для авто-перезапуска при изменениях:
npm run dev

# 3. Открыть в браузере
# http://localhost:3000

# Друзья в той же WiFi-сети заходят по вашему IP:
# http://192.168.X.X:3000  (узнать: ipconfig / ifconfig)
```

## ☁️ Деплой бесплатно

### Railway (рекомендую — самый простой)
1. Зарегистрируйтесь на https://railway.app
2. New Project → Deploy from GitHub repo (или Upload folder)
3. Railway сам найдёт `package.json` и запустит `npm start`
4. Получите URL вида `https://ваш-проект.up.railway.app`
5. Все могут заходить по этому URL 🎉

### Render
1. https://render.com → New Web Service
2. Подключите GitHub или загрузите код
3. Build command: `npm install`
4. Start command: `node server.js`
5. Free tier — сервис "засыпает" через 15 мин бездействия (первый запрос медленный)

### Fly.io
```bash
npm install -g flyctl
flyctl launch    # следуйте инструкциям
flyctl deploy
```

## 🎮 Как играть

1. **Ведущий** открывает сайт, вводит имя → **Создать лобби**
2. Получает **5-значный код** (напр. `XK7PQ`)
3. **Друзья** заходят на тот же URL, вводят имя и код → присоединяются
4. Ведущий вводит **секретное слово** и выбирает **предателя**
5. Каждый игрок тайно смотрит свою роль на своём экране
6. Игроки по очереди описывают слово — предатель слушает и пытается угадать
7. Голосование → победа мирных или предателя
