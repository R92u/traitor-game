# 🕵️ ПРЕДАТЕЛЬ — Мультиплеер

Социальная игра на доверие для нескольких устройств.

## Структура проекта

```
traitor-game-final/
├── server.js          ← Node.js сервер (Socket.io)
├── package.json       ← Зависимости
├── README.md          ← Этот файл

    └── index.html     ← Полный фронтенд
```

## 🚀 Запуск локально

### Требования
- Node.js 18+ ([скачать](https://nodejs.org))

### Шаги

```bash
# 1. Установить зависимости
npm install

# 2. Запустить сервер
npm start

# Откроется на http://localhost:3000
```

Для разработки с авто-перезагрузкой:
```bash
npm run dev
```

## 🎮 Как играть

1. **Ведущий** открывает сайт и нажимает "Создать лобби"
2. Получает **5-значный код** (например: XK7PQ)
3. **Друзья** заходят на сайт, вводят свой ник и код → присоединяются
4. Ведущий вводит **секретное слово** и выбирает **предателя** (случайно или вручную)
5. Каждый игрок берёт телефон и смотрит свою **карточку ролей**:
   - **Мирный** — видит слово
   - **Предатель** — видит "???"
6. Игра начинается:
   - Игроки по очереди нажимают "✓ Закончить ход"
   - Предатель может нажать "🎯 Угадать слово"
   - Справа кнопка "🗳️ ГОЛОСОВАНИЕ" — голосуют за исключение предателя
7. Победа:
   - **Предатель** угадал слово → **+2 очка**
   - **Мирные** выгнали предателя → **+1 очко каждому мирному**
8. Ведущий может сразу начать новую игру без переборки лобби

## ☁️ Деплой на Railway (бесплатно)

1. Зарегистрируйся на [railway.app](https://railway.app)
2. Создай GitHub репозиторий и загрузи файлы
3. В Railway: **New Project → Deploy from GitHub repo** → выбери свой репо
4. Railway автоматически:
   - Найдёт `package.json`
   - Установит зависимости (`npm install`)
   - Запустит сервер (`npm start`)
5. Получишь URL вида: `https://traitor-game-xyz.up.railway.app`
6. Отправь эту ссылку друзьям!

### Переменные окружения
Railway автоматически устанавливает `PORT` через переменные окружения.

## 📝 Особенности

✅ Полный мультиплеер на WebSocket (Socket.io)  
✅ Игровой стол с фигурками игроков  
✅ Раздача ролей поочередно (карточки видит только игрок)  
✅ Система очков между раундами  
✅ Ведущий видит все роли, не участвует в игре  
✅ Голосование с подсчётом в реальном времени  
✅ Угадывание слова только для предателя  
✅ Красивый UI с тёмной темой  

## 🔧 Технологии

- **Node.js + Express** — сервер

- **Vanilla JS** — фронтенд
- **CSS3** — стили и анимации
